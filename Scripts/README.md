# ⚙️ Active Directory Automation Scripts

## 📌 Overview

I created PowerShell scripts to automate user management and administrative tasks in Active Directory.

---

## 👤 User Creation Script

Creates a single user in a specified OU.

---

## 📊 Bulk User Script

Imports users from a CSV file and creates accounts automatically.

---

## 🔐 Password Reset Script

Resets user passwords securely using PowerShell.

---

## 🔓 Account Unlock Script

Unlocks locked accounts and verifies status.

---

## 🖥️ Group Policy Update

Forces policy updates on client machines.

---

## 🎯 Purpose

* Automate repetitive tasks
* Improve efficiency
* Reduce manual errors
* Simulate real system administration workflows
